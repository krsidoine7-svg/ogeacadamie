# 📖 Index des Documents de Spécifications (`prompt/`)

Ce fichier est l'index centralisé de tous les documents de spécifications du projet, contenus dans le dossier `prompt/`.

**ChefsOge** doit le consulter au démarrage de chaque session pour connaître les spécifications disponibles.

---

> **Note** : Ce tableau sera mis à jour au fur et à mesure que de nouveaux documents seront ajoutés dans `prompt/`. Si un document est manquant, ChefsOge doit le signaler à l'utilisateur.

---

## Comment Ajouter un Document

1. Placer le fichier `.md` dans le dossier `ChefsOge/prompt/`.
2. Mettre à jour ce fichier (`docs_index.md`) en ajoutant une nouvelle ligne dans le tableau ci-dessus.
3. Référencer le document dans les briefs de délégation si pertinent.
